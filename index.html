SNS-1

SNS-1は、硝子作家ヨウが自身のガラス作品の制作動画をTikTokへ投稿するために使用する、個人利用のSNS投稿支援ツールです。

できること

手元のパソコンから、自分のTikTokアカウントへ動画を直接投稿します（TikTok Content Posting API / Direct Postを利用）。

運営者

硝子作家ヨウ（Glass artist You）
ガラス作品・トンボ玉の制作販売を行っています。
